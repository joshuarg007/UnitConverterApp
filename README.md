# UnitConverterApp

UnitConverterApp is a simple Android application built with Jetpack Compose that converts values between different units such as temperature and length. It allows users to select conversion categories and units dynamically and view real-time results.

## Features

- Convert between Celsius, Fahrenheit, and Kelvin
- Convert between Meters, Kilometers, Feet, and Miles
- Built with Jetpack Compose for a modern, declarative UI
- Includes unit tests for conversion accuracy using JUnit

## Project Structure

- MainActivity.kt: Entry point with Compose UI
- ConversionUtils.kt: Contains all conversion logic
- ConversionUtilsTest.kt: Unit tests for verifying logic

## Requirements

- Android Studio Flamingo or newer
- Kotlin 1.9+
- Minimum SDK: 24 (Android 7.0 Nougat)

## Running Unit Tests

1. Open the project in Android Studio
2. Navigate to `ConversionUtilsTest.kt` inside `src/test/...`
3. Right-click the file and select "Run Tests"
